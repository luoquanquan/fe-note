# 笔记项目

## 初始化写作环境

- clone 本项目 `git clone https://github.com/luoquanquan/fe-note.git`
- `cd fe-note`
- enjoy

## 常用命令

- `yarn` 安装依赖
- `yarn build` 创建静态资源
- `yarn clean` 清除之前生成的网站数据
- `yarn deploy` 把当前生成的静态资源部署到远端
- `yarn server` 本地启动静态服务, 适合编写边预览
  - 本地预览需要添加 hosts: `127.0.0.1 fe-note.niubishanshan.top`
- `yarn release` 根据最新的 blog 生成静态资源站 & 并将其部署到 github
